# הגדרת קוד מנהל ב-Netlify

לאחר העלאת הפרויקט ל-GitHub/Netlify, יש להוסיף ב-Netlify משתני סביבה:

1. היכנס ל-Site configuration.
2. פתח Environment variables.
3. הוסף `ADMIN_CODE` ובחר קוד מנהל חזק.
4. הוסף `ADMIN_TOKEN_SECRET` עם ערך ארוך ואקראי (מומלץ לפחות 32 תווים).
5. ודא שקיימים גם משתני האחסון שהיו בפרויקט: `NETLIFY_SITE_ID`/`SITE_ID` ו-`NETLIFY_TOKEN`/`NETLIFY_ACCESS_TOKEN`.
6. בצע Deploy מחדש.

## הרשאות

- צפייה, פילטרים, דוחות וייצוא: פתוח לכל מי שיש לו גישה לאתר.
- הזנה ידנית, טעינת קובצי אריזה ואיכות, שינוי יעדים, הוספת אירועים, מחיקה ואיפוס: מנהל בלבד.
- כניסת המנהל נשמרת רק בכרטיסיית הדפדפן ועד 8 שעות לכל היותר.

אין לשמור את קוד המנהל בתוך index.html או ב-GitHub.
