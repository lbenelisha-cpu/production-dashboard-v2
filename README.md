# Production Dashboard V4.0

גרסה זו מבוססת על V3.9 עם האחסון המשותף התקין.

תיקונים:
- יישור נתונים מתחת לכותרות בטבלת פערי הזמנות.
- טבלת פירוט מנות איכות עם עמודות קבועות והשלמת נתונים מנתוני הייצור.
- שמירת פרטי איכות מלאים בענן, ולא רק UD Code.
- הצגת טקסט מלא בטבלת פירוט הרשומות ללא חיתוך בשלוש נקודות.
- התאמת רוחבי עמודות, גלישת שורות ו-tooltip לערכים ארוכים.

לאחר העלאת הגרסה יש לטעון פעם אחת מחדש את קובץ האיכות במחשב מנהל, כדי לשמור בענן את התיאורים המלאים.


## V4.3
Quality data is saved in multiple Netlify Blobs chunks to support large Excel files.


## V4.4
- סיכום ופירוט מנות האיכות מסוננים לפי יום, חודש, רבעון, שנה או טווח מותאם.
- סינון האיכות מכבד מתקן יחיד, מספר מתקנים או כל המתקנים.
- במתקן 1542 נשמר כלל ZFIN גם בסינון האיכות.

## V4.6
- הוספת אירועי איכות, בטיחות וסביבה פתוחה לכל המשתמשים ללא קוד מנהל.
- שמירת האירוע מתבצעת דרך פעולה ייעודית ומוגבלת ב-Netlify Function.
- פעולות ניהול, טעינת קבצים, מחיקה ושינוי יעדים נשארו מוגנות בקוד מנהל.


## V4.6
גרסת אופטימיזציה לקריאות Netlify: קריאה מרוכזת של מקטעי איכות וחודשי ייצור, Cache קצר ורענון ידני שעוקף Cache.


## טעינת איכות ב-V4.6
יש לבחור יחד בדיוק שני קבצים: קובץ סיכום איכות הכולל UD Code וקובץ תוצאות בדיקה רחב. המערכת מזהה את הקבצים לפי העמודות ולא לפי שמות הקבצים. לאחר הטעינה, לחיצה כפולה על מנה בטבלת פירוט האיכות פותחת את כל תוצאות החריגה שלה.
