# לוח בקרה — ייצור ואריזה (גרסה שיתופית ל-Netlify)

הקובץ הזה כולל את הדשבורד עצמו (`index.html`) ופונקציית שרת קטנה (`netlify/functions/storage.js`)
שגורמת לכל מי שנכנס לקישור לראות **את אותם הנתונים** (רשומות, יעדי ייצור/אריזה, איכות, אירועים) —
לא עותק נפרד לכל דפדפן.

אין הגנת סיסמה — כל מי שיש לו את הקישור יכול לראות ולערוך את הנתונים.

## איך מעלים את זה ל-Netlify (הדרך הקלה — בלי Git)

1. נכנסים ל- https://app.netlify.com ונרשמים / מתחברים (אפשר עם Google, בחינם).
2. במסך הבית לוחצים **Add new site → Deploy manually** (או "Sites" ואז גרירת תיקייה).
3. גוררים את **כל התיקייה הזו** (לא רק את index.html — חשוב לגרור גם את `netlify.toml`, את `package.json`
   ואת תיקיית `netlify/functions`) לתוך אזור ה-Drop.
4. Netlify יבנה את האתר אוטומטית (יתקין את `@netlify/blobs` ויפרסם את הפונקציה). זה לוקח כדקה.
5. מקבלים כתובת כמו `https://your-site-12345.netlify.app` — זו הכתובת שאתם משתפים עם כל מי שצריך
   להשתמש בדשבורד.

> שימו לב: כדי שהפריסה "תזכור" אתכם לעדכונים עתידיים בלי לגרור תיקייה כל פעם, מומלץ בהמשך לחבר את
> האתר לריפו ב-GitHub (Netlify → Site settings → Build & deploy → Link repository), אבל זה לא חובה
> כדי להתחיל לעבוד.

## איך בודקים שזה עבד

נכנסים לכתובת שקיבלתם. למעלה, מתחת לכותרת "לוח בקרה — ייצור ואריזה", אמורה להופיע שורה ירוקה:

> 🟢 מצב שיתופי — כל המשתמשים רואים נתונים אלו

אם מופיעה שורה צהובה "🟡 גיבוי מקומי בלבד" — סימן שהפונקציה לא עלתה כצריך (ראו "פתרון תקלות" למטה).

## פתרון תקלות

- **השורה הצהובה מופיעה גם אחרי פריסה ב-Netlify:** בדקו ב-Netlify → Functions שהפונקציה `storage`
  מופיעה ברשימה. אם לא — בדקו ב-Deploys → הלוג של ה-Build שה-`npm install` רץ בהצלחה (התקנת
  `@netlify/blobs`).
- **פתיחת הקובץ ישירות מהמחשב (לחיצה כפולה על index.html):** זה *לא* יעבוד במצב שיתופי — Netlify
  Functions רצות רק כשהאתר מפורס בפועל ב-Netlify. במצב הזה הדשבורד יחזור אוטומטית לגיבוי מקומי
  (localStorage) כדי שלא יישברו דברים, אבל זה לא יהיה משותף.
- **רוצים בכל זאת הגנת גישה בעתיד:** אפשר להוסיף הגנת סיסמה בסיסית (Netlify → Site settings →
  Visitor access → Password protection) בלי לשנות שורת קוד.

## מבנה הקבצים

```
index.html                     — האתר/הדשבורד עצמו
netlify.toml                   — קונפיגורציית הבנייה לנטליפיי
package.json                   — תלות בספריית Netlify Blobs
netlify/functions/storage.js   — פונקציית השרת שקוראת/כותבת ל-Blob Store המשותף
```
