# Production Dashboard V3.9

גרסה זו משתמשת בספרייה הרשמית `@netlify/blobs` לשמירה משותפת.

## משתני סביבה נדרשים
- `ADMIN_CODE`
- `ADMIN_TOKEN_SECRET`
- `NETLIFY_SITE_ID` — ה-Project ID
- `NETLIFY_TOKEN` — Netlify Personal Access Token

לאחר העלאה ל-GitHub יש להמתין ל-Deploy חדש, לטעון נתונים במחשב המנהל, ואז לרענן במחשב השני.
