# IML Production Dashboard V4.6.4

## שינויים

- טעינת איכות תומכת כעת בקובץ Excel מאוחד אחד הכולל שני גיליונות: גיליון סיכום איכות וגיליון תוצאות בדיקה.
- נשמרה אפשרות לטעון שני קובצי איכות יחד, לצורך תאימות לתהליך הקיים.
- זיהוי הגיליונות מתבצע לפי מבנה העמודות ולא לפי שם הקובץ או שם הגיליון.
- נוספה מניעת כפילויות בעת טעינת נתוני איכות.
- עודכנו שמות קובצי אייקון IML כדי לעקוף מטמון ישן במחשבים שבהם עדיין הופיעה האות ל.
- עודכנו manifest, Service Worker וכותרות Cache ב-Netlify.
- מספר הגרסה עודכן ל-V4.6.4.

## קובץ איכות מאוחד

הקובץ המאוחד צריך להכיל שני גיליונות באותו Workbook:

1. סיכום איכות — Batch יחד עם UD Code או QA Status.
2. תוצאות בדיקה — Batch Number, עמודות הזיהוי ועמודות תוצאות הבדיקות.

ניתן לבחור גם את שני הקבצים הישנים יחד. האפליקציה תזהה אוטומטית את המבנה.
